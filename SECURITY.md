# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing **security@patabuga.co**.

We will acknowledge your report within 48 hours and provide a detailed response within 7 days.

## Security Best Practices

- All infrastructure is managed via Terraform (Infrastructure as Code)
- Zero Trust Architecture with identity-centric access control
- All secrets managed via Cloudflare Access and Vaultwarden
- Regular security audits and compliance checks (ISO/IEC 27001:2022)
